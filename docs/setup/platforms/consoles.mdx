To set up your game for console platforms, please refer to the documentation in the each platform's Git repository. That documentation will instruct you on how to set up each platform.

## Getting access to console platforms

To get access to console platforms, please follow [the instructions in the License Manager](https://licensing.redpoint.games/organisation/console-access).

## Console support questions

If you have support questions relating to console platforms, please ask them in the console-specific Discord channel once you have verified your console access. **Do not discuss console platforms in the public Discord channels, as this is a violation of the non-disclosure agreements you have signed.**

## Not receiving the console verification email?

If you've followed the instructions in the License Manager, but are still not receiving the console access verification email after about 15 minutes, please try adding the following email address to your allowlist:

```
licensing@rdn.redpoint.games
```

:::danger
Some platforms like Outlook are known to not deliver our console verification emails, even to the spam folder, unless you add the licensing email address to your allowlist.
:::
