### 2026.06.18

This release adds support for the stable release of Unreal Engine 5.8:

- Added support for Unreal Engine 5.8 stable.
  - Free Edition users will need to update their engine version when updating to this release.
- Added a blueprint wrapper for the "On Connection Status Changed" event.
- Fixed an issue where a Development client would not have the public key option in the connection string when connecting to a Shipping server.
