:::danger
The plugin versions below are unsupported and likely have critical bugs, so no support is offered for them. In addition, many of the older versions target a version of the EOS SDK you can no longer download, or will no longer work with the EOS backend.

If you are using an engine version listed here, you will need to [upgrade your engine and plugin version](./engine_support_policy.mdx) to receive support, in accordance with our engine support policy.
:::
